# Chinmay Mahananda - Professional ECE & ML Systems Portfolio

A luxury-tier, highly interactive portfolio website for Chinmay Mahananda, engineered with a minimalist **Porsche Design aesthetic (Porsche White & Platinum Silver)** and inspired by the **Google Antigravity** interface. 

Designed specifically to showcase core competencies in **Electrical & Computer Engineering, VLSI Design, Systolic Array hardware accelerators, and Machine Learning Systems** for target roles at **Google TPU / Silicon teams**.

Live URL: [https://chinmaymahananda.github.io](https://chinmaymahananda.github.io)

---

## 🏎️ Key Features

- **Google Antigravity Torus (Porsche Edition)**: Modular concentric 3D rings rotating on multiple axes around a central floating silicon accelerator chip.
- **Dynamic Silicon Particle Canvas**: A high-fidelity interactive canvas background drawing orthogonal (L-shaped) trace networks between floating coordinates to simulate PCB routing traces.
- **Interactive Systolic Matrix Skill Visualizer**: A 4x4 processor matrix representing engineering competencies. Hovering over nodes propagates mock row/column data bus waves to trigger detailed latency/clock-cycle readouts.
- **Google-Readiness Terminal**: A simulated interactive Antigravity TUI terminal running diagnostics checks, displaying RTL/Verilog compilation streams, testbench runs, and outputting candidate fit levels.
- **Interactive Hardware Simulators**:
  - **8-Bit ALU Simulator**: Computes bitwise operations (`ADD`, `SUB`, `AND`, `OR`, `XOR`) and outputs flag indicators (`Zero`, `Carry`, `Negative`).
  - **INT8 CNN MAC Stepper**: Multiplies and accumulates inputs ($Acc += X \times Y$) through simulated registers.
  - **PCB Bring-Up Pipeline**: Steps users through physical board debugging and signal troubleshooting.
  - **YOLO11 OBB Scanning Camera**: Mock scans and matches cards via triplet embeddings.

---

## 🛠️ Tech Stack & Architecture

- **Core**: Semantic HTML5, Vanilla JavaScript (ES6+), HTML5 Canvas API.
- **Styles**: Custom Vanilla CSS3 using HSL design tokens, blur-filter glassmorphism, responsive grid overlays, and custom keyframe transition behaviors.
- **Design Inspiration**: Google Antigravity & Porsche E-Performance minimalist branding.
- **SEO Optimization**: Configured with meta descriptors, social graph tags, and responsive viewport parameters.

## ⚙️ How to Run Locally

To test the portfolio locally without installing any third-party dependencies, open a terminal in this directory and launch a lightweight web server:

```bash
# Using Python 3
python -m http.server 8080

# Using Node (npx)
npx http-server -p 8080
```

Open your browser and navigate to `http://localhost:8080` to view the site.
